# CrawlerDesafio — Guia para agentes

## O que é

Web scraper em Node.js + TypeScript para extrair livros de [books.toscrape.com](https://books.toscrape.com/).

## Como trabalhar neste repositório

1. Ler `.cursor/rules/` — missão, fases e stack estão definidas lá.

## Fases (resumo)

1. Bootstrap (TS, ESLint, Jest, estrutura)
2. Scraping com Playwright
3. Testes unitários
4. Bônus: parsing com IA (OpenAI/Gemini)
5. Bônus: JSON/CSV
6. Bônus: PostgreSQL + docker-compose

## Comandos (após bootstrap)

Serão documentados no `README.md` à medida que as fases forem concluídas (`npm test`, `npm run lint`, `npm run scrape`, etc.).
